# WS5 Smart Forms

A semantic, accessible, validated smart form including custom validation, autosave, phone normalisation, and an error summary.

## Features
- Semantic HTML5 form with labels, fieldsets, native validation
- Visual validation states (valid/invalid)
- Custom JavaScript validation
- Autosave using localStorage
- Phone number normalisation on blur
- Error summary with focus management
- Honeypot bot detection
